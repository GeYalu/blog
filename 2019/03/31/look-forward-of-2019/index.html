Look Forward of 2019 Q2

本文章是look forward系列的第一篇，从Q2开始，计划每个Q开始的时候做一次总结，指定一次OKR，用OKR的方式来规划自己的生活目标，第一次尝试，话不多说，进入正题。

2019 Q2 OKR：
Timeline: 2019.4.1-2019.7.1
Objective:

提升沟通能力

Key Results:

1.完成易灵微课中“沟通口才”中的两门课；

2.读完至少一本沟通的书：非暴力沟通、沟通的艺术；

3.每周有一次与朋友的约饭沟通；

4.每周与家人沟通一次；

5.每天与女朋友有效沟通一次；

Objective:

提升技术能力

Key Results:

1.读完5本技术类书籍；

2.完整学完一次线上课程；

3.完成90道面试题的记录；

4.完成coursera算法课第一部分；

Objective:

提升认知水平

Key Results:

1.每周写一篇自己思考的文章；

2.选择10个优秀的内容渠道，保持跟踪阅读；

Objective:

Keep Healthy

Key Results:

1.每周运动两次；

2.每周至少有3天吃早饭；

3.每周至少吃一次水果；

Go away, fu* 2018.
